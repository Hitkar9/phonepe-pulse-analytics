# PhonePe Pulse: Transaction Growth & User Analytics

## Project Overview
End-to-end Product Analytics project analyzing PhonePe's 
real public Pulse dataset combined with 50,000 synthetic 
users built on NPCI/UPI industry benchmarks.

## Business Questions Answered
- Which states are driving PhonePe's growth?
- Where does the user funnel drop off?
- Does cashback improve user retention?
- Which user segments churn most?
- What is PhonePe's market opportunity in underpenetrated states?

## Key Findings
- PhonePe grew **91x** in transactions from 2018 to 2024
- **2021 spike** driven by post-COVID digital payments boom
- Biggest funnel drop-off: **KYC → Bank Link** (22.5% drop)
- Cashback users do **73% more transactions** than non-cashback
- **Bihar and UP** have high ARPU but low penetration = opportunity
- Power users (top 10%) spend **42x more** than bottom 10%
- Overall churn rate: **19.44%** | Jharkhand highest at 21.17%

## Tech Stack
- **Database:** Microsoft SQL Server
- **SQL Tool:** SSMS (SQL Server Management Studio)
- **Python:** pandas, numpy, matplotlib, seaborn, sqlalchemy
- **IDE:** VS Code + Jupyter Notebook
- **Data Source:** PhonePe Pulse (real) + Synthetic data (NPCI benchmarks)
